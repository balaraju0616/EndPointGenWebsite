HOW TO USE:
-----------
1. Double-click index.html  (or EndPointGen_Demo_Standalone.html)
2. Click ▶ AUTOPLAY for hands-free playback
3. Arrow keys ← → to navigate manually
4. Press P to pause/resume
5. Click dots at the bottom to jump to any section

RECORDING AS A VIDEO:
---------------------
1. Open the file in Chrome/Edge
2. Press F11 for fullscreen
3. Click AUTOPLAY
4. Record with OBS Studio or Windows Game Bar (Win+G)
   → Total runtime: approximately 2 minutes
